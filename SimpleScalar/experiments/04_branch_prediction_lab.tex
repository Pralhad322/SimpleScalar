\section{Objective}
This lab studies the effect of branch prediction schemes on branch address prediction rate and processor CPI/IPC. The lab uses \texttt{sim-bpred} for branch predictor statistics and \texttt{sim-outorder} for timing impact.
The structure follows the UMass SimpleScalar branch prediction lab, with paths updated for the current repository layout:
\begin{lstlisting}
http://www.ecs.umass.edu/ece/koren/architecture/Simplescalar/branch_prediction.htm
\end{lstlisting}
Run all commands from the cloned repository root.
\begin{lstlisting}
cd /path/to/SimpleScalar
\end{lstlisting}
\section{Setup}
Configure for Alpha because this lab uses the Alpha benchmarks from Lab 1:
\begin{lstlisting}
make clean
make config-alpha
make
mkdir -p labs
\end{lstlisting}
Get help for \texttt{sim-bpred}:
\begin{lstlisting}
./sim-bpred -h
\end{lstlisting}

\begin{lstlisting}

\section{Part 1: Branch Address Prediction Rate}

\begin{lstlisting}
