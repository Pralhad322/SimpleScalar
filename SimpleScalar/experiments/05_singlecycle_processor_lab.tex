\section{Objective}
This lab uses the course simulator \texttt{sim-singlecycle} to demonstrate a single-cycle execution model. The simulator executes instructions functionally like \texttt{sim-safe}, but it charges exactly one simulated cycle per committed instruction.
Important relationship:
\begin{lstlisting}
CPI = total cycles / total instructions
\end{lstlisting}
For this educational single-cycle model:
\begin{lstlisting}
sim_cycle = sim_num_insn
sim_CPI = 1.0000
\end{lstlisting}
\section{Setup}
Run from the cloned repository root:
\begin{lstlisting}
cd /path/to/SimpleScalar
make clean
make config-pisa
make
mkdir -p labs
cp tests-pisa/bin.little/test-math labs/
\end{lstlisting}
Confirm that \texttt{sim-singlecycle} exists:
\begin{lstlisting}
ls sim-singlecycle
\end{lstlisting}
\section{Part 1: Run the Single-Cycle Simulator}
Run:
\begin{lstlisting}

\begin{lstlisting}
