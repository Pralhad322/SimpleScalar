This folder contains the student experiment handouts for the SimpleScalar computer architecture lab sequence. Each experiment is provided as a PDF for students and as a LaTeX source file for editing.
Repository layout after cloning:
\begin{lstlisting}
SimpleScalar/
  benchmarks/
  config/
  labs/
  scripts/
  SimpleScalar/
    experiments/
  tests-alpha/
  tests-pisa/
  sim-fast.c
  sim-safe.c
  sim-profile.c
  sim-bpred.c
  sim-outorder.c
  sim-singlecycle.c
  Makefile
\end{lstlisting}
There is no nested \texttt{simplesim-3.0/} directory in this repository. Unless a lab explicitly says to enter \texttt{labs/}, run commands from the cloned repository root:
\begin{lstlisting}
cd /path/to/SimpleScalar
\end{lstlisting}

\section{Notes}
\begin{itemize}
\item Lab 1 uses Alpha and PISA builds. Rebuild with \texttt{make clean} before switching ISA targets.
\item Labs 2, 4, and 5 use PISA.
\item Lab 3 uses Alpha benchmarks for branch prediction, matching the original UMass SimpleScalar reference lab.
\item The local PISA cross compiler is installed under \texttt{toolchain/} by \texttt{scripts/install\_pisa\_cross\_compiler.sh}; it is not committed to the repository.
\end{itemize}
